Data Structures Quiz

1. Which data structure uses LIFO (Last In First Out) principle?

Correct Answer: Stack

2. Which data structure uses FIFO (First In First Out) principle?

Correct Answer: Queue

3. What is the time complexity of accessing an element in an array by index?

Correct Answer: O(1)

4. Which data structure is best suited for searching, inserting, and deleting items in a sorted list?

Correct Answer: Binary Search Tree

5. Which data structure is primarily used to implement priority queues?

Correct Answer: Heap

6. Which data structure consists of nodes where each node contains data and a pointer to the next node?

Correct Answer: Linked List

7. Which data structure stores data in key-value pairs for fast lookup?

Correct Answer: Hash Table

8. Which data structure is used to represent networks, like social networks or maps?

Correct Answer: Graph

9. Which tree traversal method visits nodes in ascending order in a BST?

Correct Answer: Inorder

10. What is the time complexity to search an element in an unsorted array?

Correct Answer: O(n)
