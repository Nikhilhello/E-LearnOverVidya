Amazon S3 & Cloud Computing Quiz

1. What does S3 stand for in Amazon S3?

Correct Answer: Simple Storage Service

2. Which of the following is a key feature of Amazon S3?

Correct Answer: Data Durability

3. What is the maximum size of a single object that can be stored in Amazon S3?

Correct Answer: 5 TB

4. What is the name of the storage container in Amazon S3?

Correct Answer: Bucket

5. Which tool is commonly used to manage Amazon S3?

Correct Answer: AWS Management Console

6. What is cloud computing?

Correct Answer: Delivering computing services over the internet

7. Which of the following is a primary benefit of cloud computing?

Correct Answer: Scalability

8. What is the most common deployment model for cloud computing?

Correct Answer: Public Cloud

9. What term describes the ability to scale resources up or down based on demand?

Correct Answer: Elasticity

10. What is the billing model where you pay only for the resources you use in cloud computing?

Correct Answer: On-Demand Pricing
